# Pupper Client

A Better, Faster Minecraft Client for 1.21.4

## Features
- Optimized performance and gameplay experience
- Custom HUD and UI elements
- Built-in ViaFabricPlus for multi-version support
- In-game account switching capability
- Enhanced rendering and visual effects

## Installation

### Method 1: Direct Installation
1. Download the mod file from our official sources
2. Drag and drop the mod file into PCL or HMCL launcher

### Method 2: Modpack Installation
1. Download the complete modpack from our official sources
2. Drag and drop the modpack into PCL or HMCL launcher

## Requirements
- **Minecraft**: 1.21.4
- **Required Mods**:
    - [IAS] In-Game Account Switcher
    - ViaFabricPlus
